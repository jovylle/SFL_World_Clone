Land boosts API (beta)
GET
https://sfl.world/api/v1/land/114779
The average value for AOE NFTs is calculated for the affected beds relative to the total amount of beds
For example, Queen Cornelia can give you +1 Corn, but only for 10 beds max, so min=1, max=2, but on average for 50 beds = ((40 x 1) + (10 x 2)) / 50 = 1.2
